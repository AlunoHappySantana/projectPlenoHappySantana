## 🎯 Objetivo
O objetivo do projeto é desenvolver um jogo utilizando uma **ODS**.  
Escolhemos a:

> **ODS 9 — Indústria, Inovação e Infraestrutura**

Nos inspiramos no jogo *Papa's Pizzeria*, porém com uma adaptação:
- Em vez de uma pizzaria 🍕  
- Criamos uma **pastelaria 🥟**
